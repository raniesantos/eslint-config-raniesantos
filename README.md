My personal ESLint configuration.

___
## Install

Install this package as a devDependency:

```bash
$ npm install --save-dev eslint-config-raniesantos
```

Then have your project's `.eslintrc` file extend the ruleset.

```json
{
    "extends": ["raniesantos"]
}
```
